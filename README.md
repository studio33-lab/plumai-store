## ğŸ› ï¸ Manual Installation
*Ideally, you should install these packages directly inside the PlumAi application.*
However, if you are offline or an advanced user (Bioengineer level):

1.  Download the `.zip` of the package from the `packages/` directory.
2.  Extract it to your local PlumAi data folder:
    * **Windows:** `%AppData%/PlumAi/packages/`
    * **Linux:** `~/.local/share/plumai/packages/`
3.  Restart PlumAi.
